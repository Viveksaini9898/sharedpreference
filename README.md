# SharedPreferenceLibrary


# Install Dependency

If you haven't already, add the jitpack repository to the root build.gradle file.

# 
    allprojects {
       repositories {
           ...
           maven { url 'https://jitpack.io' }
       }
    }
    
 # Add the dependency:

    	        implementation 'com.github.viveksaini75:sharedpreference:1.0.1'
